Description

At dawn atop Mount Bromo, Pak Dengklek is playing with a sequence while enjoying the sunrise.

Pak Dengklek has a sequence of integers $(A_0, A_1, \ldots, A_{N-1})$ of length $N$. The length $N$ is of the form $2^k$ where $k$ is a positive integer.

Pak Dengklek wants to do $Q$ operations, each of which is one of the following three types.

  1. 1 X: Sequentially from $i = 0$ to $i = N - 1$, if $i < (i \oplus X)$, then swap elements $A_i$ and $A_{i \oplus X}$. The $\oplus$ is a bitwise XOR operator.
  2. 2 L R: Calculate $A_L + A_{L+1} + \ldots + A_R$.
  3. 3 L R: Calculate $A_L \times 1 + A_{L+1} \times 2 + \ldots + A_R \times (R - L + 1)$.

For each operation of type $2$ and $3$, Pak Dengklek wants to know the result. Help him!

Notes

Bitwise XOR of two integers $x$ and $y$ can be computed as follows.

  1. Write $x$ and $y$ in base two with the same bit length.
  2. Perform the $\text{XOR}$ operation for each corresponding bit pair:
    • $0 \text{ XOR } 0 = 0$
    • $0 \text{ XOR } 1 = 1$
    • $1 \text{ XOR } 0 = 1$
    • $1 \text{ XOR } 1 = 0$
  3. Convert the $\text{XOR}$ result back to base ten.

As an example, $6 \text{ XOR } 10 = 12$. Because $6 = 0110$ and $10 = 1010$ in base two, the bitwise XOR obtained is $1100 = 12$ in base ten.

Constraints

  • $2 \le N \le 2^{19}$
  • $1 \le Q \le 200\,000$
  • $N = 2^k$ for some positive integer $k$.
  • $1 \le A_i \le 1\,000\,000$ for each $0 \le i \le N - 1$.
  • $0 \le X \le N - 1$ for each operation of type $1$.
  • $0 \le L \le R \le N - 1$ for all operation of types $2$ and $3$.
  • There is at least one operation of type $2$ or $3$.

Subtasks

  1. (6 points) $N \le 2^{10}$; $Q \le 1000$
  2. (8 points) No operations of type $1$.
  3. (10 points) All operations of type $1$ happen before operations of type $2$ or $3$.
  4. (20 points) There are no operations of type $3$.
  5. (17 points) $N \le 2^{16}$
  6. (19 points) $L = 0$ and $R = N - 1$ for all operation of types $2$ and $3$.
  7. (20 points) No additional constraints.

Input

The input is given in the following format: 

N Q
A0 A1 … AN-1
Operation1
Operation2
⋮
OperationQ

Output

For each operation of type $2$ and type $3$, output the result in one line.

Sample Input

8 5
1 2 3 4 5 6 7 8
1 3
2 1 6
1 7
2 0 7
3 2 7

Sample Output

27
36
73

Explanation

In the first operation, the pair of indices that were swapped are $(0, 3)$, $(1, 2)$, $(4, 7)$, $(5, 6)$, so the sequence $A$ became $(4, 3, 2, 1, 8, 7, 6, 5)$.

In the second operation, Pak Dengklek wanted to count $A_1 + A_2 + A_3 + A_4 + A_5 + A_6 = 3 + 2 + 1 + 8 + 7 + 6 = 27$.

In the third operation, the pair of indices that were swapped are $(0, 7), (1, 6), (2, 5), (3, 4)$, so the sequence $A$ became $(5, 6, 7, 8, 1, 2, 3, 4)$.

In the fourth operation, Pak Dengklek wanted to count $A_0 + A_1 + A_2 + A_3 + A_4 + A_5 + A_6 + A_7 = 5 + 6 + 7 + 8 + 1 + 2 + 3 + 4 = 36$.

In the fifth operation, Pak Dengklek wanted to count $A_2 \times 1 + A_3 \times 2 + A_4 \times 3 + A_5 \times 4 + A_6 \times 5 + A_7 \times 6 = 7 \times 1 + 8 \times 2 + 1 \times 3 + 2 \times 4 + 3 \times 5 + 4 \times 6 = 73$
