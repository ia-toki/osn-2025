Description

Malang is a city that frequently hosts festivals. Since the National Olympiad is being hosted in Malang, Pak Dengklek will liven up the event by preparing a new festival, which is the LCM Festival!

There are $N$ tenants participating in this festival. In this festival, Pak Dengklek asks the $i$-th tenant to mention a positive integer $A_i$. Pak Dengklek then ask the festival attendees to give a sequence $B$ of $N + 1$ positive integers, in such a way that:

  • For every index $i$ $(1 \leq i \leq N)$, $A_i$ is the Least Common Multiple ($\text{LCM}$) of $B_i$ and $B_{i + 1}$.

For instance, if the sequence $A$ obtained is $[3, 12, 12, 4]$, then one possible $B$ that satisfies the rule is $[1, 3, 12, 4, 1]$ because:

  • $A_1 = 3, B_1 = 1, B_2 = 3$. $\text{LCM}(1, 3) = 3$.
  • $A_2 = 12, B_2 = 3, B_3 = 12$. $\text{LCM}(3, 12) = 12$.
  • $A_3 = 12, B_3 = 12, B_4 = 4$. $\text{LCM}(12, 4) = 12$.
  • $A_4 = 4, B_4 = 4, B_5 = 1$. $\text{LCM}(4, 1) = 4$.

Attendees will win a prize for every distinct sequence $B$ that is given. Two sequences $B$ and $B'$ are said to be different if there exists an index $i$ ($1 \leq i \leq N+1$) such that $B_i \neq B'_i$.

To ensure the success of the festival, Pak Dengklek wants to know how many prizes he has to prepare in case all possible sequences of $B$ are given correctly. Since the number can be very large, output the result modulo $1\,000\,000\,007$.

Notes

LCM of two positive integers is defined as the smallest positive integer divisible by both integers. For instance, LCM of $4$ and $6$ is $12$, because $12$ is divisible by $4$ as well as $6$, and there is no smaller positive integer that is divisible by both integers.

Constraints

  • $1 \leq N \leq 100\,000$
  • $1 \leq A_i \leq 1\,000\,000$ for $1 \leq i \leq N$

Subtasks

  1. (8 points) $N \leq 5; A_i \leq 12$
  2. (8 points) $N = 1$
  3. (13 points) $N \leq 100; A_i \leq 100$
  4. (15 points) $N \leq 100; A_i \leq 5000$
  5. (18 points) $A_i = 2^k$ for some non-negative integer $k$
  6. (17 points) $A_i \leq 5000$
  7. (21 points) No additional constraints.

Input

The input is given in the following format:

N
A1 A2 ... AN

Output

An integer representing the number of possible sequences $B$ modulo $1\,000\,000\,007$.

Sample Input 1

4
3 12 12 4

Sample Output 1

15
